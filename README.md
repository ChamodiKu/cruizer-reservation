# Cruizer Reservation

## Intro
Cruizer reservation is a vehicle service reservation system build for Cruizer
Service Stations. 

## How To Setup

+ Clone the repository
+ Build and run backend
  1. Navigate to backend 
      ```bash
      cd _repository_/backend
      ```
  2. Install dependencies
      ```bash
      npm install
      ```
  2. Serve application
      ```bash
      npm run start
      ```
